# DOGESH – The Ultimate Pet Care App

Built by Antigravity App Development, **DOGESH** is a smart Android app that makes pet care simple. It connects busy pet owners with trusted local providers for feeding, walking, and training.

## Key Features
- **Easy Profiles**: Quick sign-up for both pet owners and service providers.
- **Local Search**: Find nearby pet helpers easily using Google Maps.
- **Simple Scheduling**: Book appointments in real-time.
- **Safe Payments**: Pay securely through the app using Razorpay.
- **Live Updates**: Fast and reliable data synced through Firebase.
- **Trusted Reviews**: Read ratings and feedback to choose the best care.
- **Booking History**: Keep track of all your past and upcoming services.

## Technology Stack
To build a smooth and reliable Android experience, we used a straightforward tech stack:

| Component | Technology |
| --- | --- |
| **Platform** | Android |
| **Backend & Database** | Firebase |
| **Location** | Google Maps API |
| **Payments** | Razorpay |
